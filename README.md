# Aluminum Frame Designer Mini Program

一个无后端的微信小程序项目，用于绘制铝型材结构，并进行 3D 展示、材料拆分、连接件计算与 BOM 导出。

目标场景：

- 衣柜铝型材框架
- 书桌 / 工作台框架
- 电视支架 / 展示架
- 定制储物架、设备架、花架、隔断框架

项目原则：

- **前端 only**：不依赖后端，所有项目数据存储在微信本地缓存。
- **参数化建模**：用户输入尺寸、层数、格口、型材规格，即可生成结构。
- **可视化优先**：2D 平面编辑 + 3D 预览，降低沟通成本。
- **自动 BOM**：根据结构节点、杆件长度、连接方式，自动计算型材、角码、螺栓、滑块螺母、封盖等数量。

## 当前项目状态

这是第一版项目骨架，重点沉淀产品需求、数据模型、计算规则和小程序目录结构。下一步可以直接进入微信开发者工具，实现编辑器和 3D 预览。

## 技术方案

- 微信小程序原生框架
- Canvas / WebGL 用于 3D 展示
- 本地缓存 `wx.setStorageSync` 保存项目
- 不接服务器、不登录、不云开发

## 核心模块

1. 项目管理
   - 新建结构项目
   - 选择模板：衣柜、书桌、货架、电视支架、自定义框架
   - 本地保存 / 复制 / 删除

2. 参数化设计
   - 宽、高、深
   - 型材规格：2020 / 3030 / 4040 / 4080 等
   - 层板数量、横梁数量、竖向分区
   - 门板 / 层板 / 桌面 / 背板作为附加件

3. 结构编辑
   - 添加杆件
   - 删除杆件
   - 调整杆件长度和位置
   - 节点吸附
   - 按 X/Y/Z 三轴生成框架

4. 3D 展示
   - 旋转、缩放、平移
   - 透视视角
   - 正视 / 俯视 / 侧视
   - 型材以方管或真实槽型简化显示

5. BOM 计算
   - 型材总长度
   - 每根切割长度
   - 连接件数量
   - 螺栓、滑块螺母、角码、端盖数量
   - 板材尺寸清单

6. 导出
   - BOM 文本复制
   - 截图分享
   - 后续可扩展 CSV / Excel 导出

## 推荐 MVP

第一阶段不要追求完整 CAD。先把产品做成一个“可视化报价和结构沟通工具”：

- 选择模板
- 输入尺寸
- 自动生成结构
- 3D 预览
- 自动生成 BOM

这就是最小闭环。它像一把尺，也像一张报价单，让设计、施工、采购之间不再各说各话。
