# 功能需求清单

## 1. 产品定位

创建一个微信小程序，用于快速绘制铝型材结构，支持 3D 展示，并自动计算材料与连接配件。

它不是重型 CAD，而是面向真实装修、定制框架、现场沟通和采购报价的轻量设计器。

## 2. 用户角色

### 2.1 个人用户

- 想自己设计衣柜、书桌、置物架、电视架。
- 需要知道大概用多少型材和配件。
- 需要拿着图纸和清单去淘宝、1688 或线下加工店沟通。

### 2.2 设计 / 装修 / 定制服务方

- 给客户快速出方案。
- 在现场根据尺寸生成结构草图。
- 快速估算成本和采购清单。

### 2.3 加工商 / 安装方

- 根据 BOM 查看切割长度。
- 根据连接方式准备角码、螺栓、滑块螺母、端盖。

## 3. 核心用户流程

1. 新建项目。
2. 选择模板：衣柜 / 书桌 / 货架 / 电视支架 / 自定义。
3. 输入外部尺寸：宽、高、深。
4. 选择型材规格：2020 / 3030 / 4040 / 4080。
5. 选择连接方式：内置连接件 / 角码 / 三通连接 / 端面攻丝。
6. 系统自动生成结构。
7. 用户进入 3D 视图检查比例。
8. 用户微调杆件、分区、层板、桌面、柜门。
9. 系统生成 BOM。
10. 用户复制清单或截图分享。

## 4. 功能模块

### 4.1 首页 / 项目管理

- 创建新项目。
- 查看本地保存项目。
- 复制项目。
- 删除项目。
- 查看最近编辑时间。

### 4.2 模板中心

模板分为：

- 衣柜框架
- 书桌框架
- 置物架
- 电视支架
- 洗衣机架
- 阳台花架
- 自定义空白框架

每个模板包括：

- 默认尺寸
- 默认型材规格
- 默认连接方式
- 默认结构逻辑
- 默认 BOM 规则

### 4.3 参数化生成

用户可以输入：

- 宽度 width
- 高度 height
- 深度 depth
- 层数 shelves
- 纵向分区 columns
- 横向分区 rows
- 是否增加背部加强杆
- 是否增加斜撑
- 是否增加脚轮 / 调平脚
- 是否增加板材：桌面、层板、柜门、背板

系统根据参数生成：

- 节点 nodes
- 杆件 beams
- 面板 panels
- 配件 accessories

### 4.4 2D 编辑

- 正视图编辑。
- 俯视图编辑。
- 侧视图编辑。
- 点击杆件查看长度。
- 点击节点查看连接方式。
- 增加 / 删除横梁。
- 增加 / 删除竖梁。
- 修改分区尺寸。

### 4.5 3D 展示

- 使用 Canvas / WebGL 渲染简化 3D 框架。
- 支持旋转、缩放、平移。
- 支持视角切换：正视、侧视、俯视、等轴测。
- 型材先用矩形柱体表达，后续再增强槽型细节。
- 点击杆件高亮显示对应 BOM 项。

### 4.6 连接方式设置

支持连接方式：

1. 角码连接
   - 每个 90° 节点默认 1-2 个角码。
   - 每个角码需要 2 套螺栓 + 滑块螺母。

2. 内置连接件
   - 每个杆端 1 套。
   - 适合外观更干净的框架。

3. 端面攻丝 + 螺栓
   - 每个杆端需要攻丝。
   - 每个连接点需要螺栓。

4. 三通 / 多通连接件
   - 适合 2020 / 3030 轻型框架。
   - 节点按连接方向数量计算。

### 4.7 BOM 计算

BOM 输出分为：

#### 型材清单

- 型材规格
- 长度
- 数量
- 总米数
- 建议购买米数，包含损耗率

#### 切割清单

- 编号
- 型材规格
- 切割长度
- 数量
- 对应结构位置

#### 配件清单

- 角码
- 内置连接件
- 螺栓
- 滑块螺母
- 端盖
- 脚轮 / 调平脚
- 板材固定件

#### 板材清单

- 桌面板
- 层板
- 柜门
- 背板
- 每块板材的宽、高、厚度、数量

### 4.8 成本估算，后续版本

- 型材单价：元 / 米
- 配件单价
- 板材单价
- 加工费：切割、攻丝、包装、运输
- 总价估算

### 4.9 导出与分享

MVP：

- 复制 BOM 文本
- 保存设计截图
- 微信分享图片

后续：

- 导出 CSV
- 导出 Excel
- 导出 PDF 报价单

## 5. 非功能需求

- 无后端。
- 所有项目数据存储在本地。
- 小程序打开速度快。
- BOM 计算逻辑必须透明，可追溯。
- 支持离线编辑。
- 数据结构要方便未来接入后端或云同步。

## 6. MVP 范围

第一版只做四件事：

1. 模板生成结构。
2. 3D 预览。
3. BOM 计算。
4. 本地保存。

不要第一版就做复杂自由建模。先让它成为一个能跑通真实采购沟通的工具。
